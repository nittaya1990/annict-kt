## Installation

### Gradle

#### build.gradle.kts
```kotlin
dependencies {
    implementation("jp.annict:annict-kt:$AnnictKtVersion")
}
```

### Maven

#### pom.xml
```xml
<dependency>
  <groupId>jp.annict</groupId>
  <artifactId>annict-kt</artifactId>
  <version>${annictKtVersion}</version>
</dependency>
```
